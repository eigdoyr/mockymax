# MockyMax

Free mockup generator. Drop a screenshot, get a premium device mockup.

> ⚠️ **Status:** Early development. Not yet usable.

## What this is

MockyMax turns screenshots into premium device mockups in seconds — no signup, no watermark. Pick a hand-curated scene (MacBook on concrete, iPhone on leather, iPad on linen), drop in your screenshot, export.

## Status

Currently building **v0.1 — Foundation**. Track progress: [Project Board](https://github.com/users/eigdoyr/projects/5)

## Development

Requires Node 22+ and pnpm 9+.

```bash
pnpm install
```

More instructions coming as the project takes shape.
